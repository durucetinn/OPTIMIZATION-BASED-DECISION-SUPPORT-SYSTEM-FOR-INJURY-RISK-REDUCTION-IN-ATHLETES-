# Clinical Decision Support and Injury Risk Optimization
This repository contains the source code for Capstone Project. 
It includes an optimization-based decision support system for athlete injury risk assessment using AHP, OWA, and Mixed-Integer Linear Programming (MILP).

## Files
* `app.py`: Streamlit dashboard code
* `data_loader.py`: Data generation, AHP, and OWA aggregation algorithms
* `optimization_models.py`: LP/MILP optimization models using PuLP
